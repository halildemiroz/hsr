# HSR Software Rendering Project

## Introduction
This is a project to understand software rendering.

## Building
After cloning the repositoriy
1. `cd tests/`
2. `make`
3. `./bin/x11-test` or `./bin/test-obj`

## Dependincies
1. Gnu Make
2. Mingw32 or MSYS2
